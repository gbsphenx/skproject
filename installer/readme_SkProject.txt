SKPROJECT
Dungeon Master ][ Skullkeep, unofficial port for Windows
========================================================
Current Release : june 2021, Sphenx
SkWin conversion code (DM2 PC9821) by Kentaro
SkullWin/SkWinDOS conversion code (DM2 PC-DOS) by Bit
Enhancements, packaging and custom graphics.dat by Sphenx


SkProject contains:
- SkWin    : Dungeon Master II Windows port from PC-9821 version
- SkullWin : Dungeon Master II Windows port from PC-DOS version
- DM2DC2   : Dungeon/Map editor for Dungeon Master II
- DM2GDED  : Graphics.dat resources editor for Dungeon Master II


For more information about SkWin/SkullWin,
please check "readme_SkWin.txt"
