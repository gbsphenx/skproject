Activator, finite activator relay

Activator works on activation until expire.

On activation, it checks if counter is expired or not. if not, it decrements and send a Message to Target.

If Cnt <> 0 Then
    QUEUE_SEND_SIGNAL
    Cnt = Cnt - 1
End If

See also Appendix: finite activator relay for working variations.

Also Delay, Beep are available.
